# CPS Embedded C Lab

This repository contains the code for the eight sessions of the CPS lab course offered by the Intelligent Embedded Systems department.

## Content

- [**`lecture_2`**](lecture_2): Functional tests for embedded hardware
- [**`lecture_3`**](lecture_3): Fundamentals of microcontroller programming using the example of LED control
- [**`lecture_4`**](lecture_4): Generating square wave signals and pulse width modulation
- [**`lecture_5`**](lecture_5): Using push buttons in embedded software
- [**`lecture_6`**](lecture_6): Embedded software using the example of simple sensor data preprocessing
- [**`lecture_7`**](lecture_7): Outputs of an embedded system on an OLED display
- [**`lecture_8`**](lecture_8): Processing sensor data with embedded AI
